# Projet VV XR360

## Installation pour Unity

### Étapes pour récupérer le projet :

1. **Décompresser le fichier ZIP**
   - Cliquez droit sur `Projet_VV_XR3607.zip`
   - Sélectionnez "Extraire vers..." ou "Extraire tout"
   - Choisissez votre dossier de destination

2. **Ouvrir le projet dans Unity Hub**
   - Lancez **Unity Hub**
   - Cliquez sur **"Ajouter"** (ou "Add")
   - Naviguez jusqu'au dossier décompressé
   
   **IMPORTANT** : Vous devez sélectionner le dossier **à l'intérieur** du dossier principal (celui qui contient les dossiers `Assets`, `ProjectSettings`, etc.)
   
   - Ne sélectionnez PAS le dossier parent (celui qui porte le nom du projet)
   - Sélectionnez directement le dossier contenant les fichiers du projet Unity

3. **Lancer le projet**
   - Double-cliquez sur le projet dans Unity Hub
   - Attendez que Unity ouvre le projet
   - C'est bon !

---

## Erreur courante

Si vous sélectionnez le mauvais dossier lors de l'ajout dans Unity Hub, vous aurez une erreur. Assurez-vous de bien passer par ces étapes :

**Correct** : `Projet_VV_XR360/Assets/` , `ProjectSettings/`, etc.

**Incorrect** : `Projet_VV_XR3607/Projet_VV_XR360/Assets/`
